# disaster_master
Nature disasters aggregation service

Service aggregates multiple nature disasters APIs (earthquakes, floods, nature fires, vulcano erruptions) and provide a one view for all of them.

# Technology used
##  Backend
* Python
* Flask
## Frontend
* React
